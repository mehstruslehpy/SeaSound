inputModes

The various input modes of this widget. Select mode is used for selecting several notes at once. Note mode is used for entering notes via mouse clicks. Delete mode is used to remove notes by individually clicking on them. Paste mode copies notes currently selected from using the select mode to the current cursor position. Remove mode deletes collections of notes which were selected via the select mode.
