Class: SliderCanvas

SliderCanvas(query, trackName, horizontalCells, verticalCells, beatsPerCell, rectangleStyle)

The slider class stores state and methods for dealing with slider canvas widgets. Slider canvas widgets are used for storing data ("sliders") to do with discrete numerical values with respect to time.

Constructor

new SliderCanvas(query, trackName, horizontalCells, verticalCells, beatsPerCell, rectangleStyle)

Construct a slider canvas widget instance and draw it to the screen.
