Class: CodedEventCanvas

CodedEventCanvas(query, trackName, cells, beatsPerCell)

The coded event class stores state and methods for dealing with coded event canvas widgets. Coded event canvas widgets are used to emit textual events with respect to time.

Constructor

new CodedEventCanvas(query, trackName, cells, beatsPerCell)

Construct a coded event canvas widget instance and draw it to the screen.

query string String containing html id of the canvas we are constructing for.
trackName string String containing the track name that this widget corresponds to.
